# LINE公式アカウント 運用マニュアル

このページは、スマホからFAQ・メニュー・お知らせを更新するための操作説明です。
専門的な知識は必要ありません。

このURLをスマホのホーム画面に追加しておくと、次回から簡単に開けます。

## 1. FAQ（よくある質問）の追加・編集・無効化

FAQは、お客様からよく届く質問への自動返信の元になる情報です。
ここに登録した内容をもとに、AIが自動で返信します。

### 1-1. FAQを追加する

1. 管理画面トップの「新規追加」ボタンをタップします。
2. 次の4つを入力します。
   - **質問**: お客様が聞いてきそうな内容（例: 「営業時間は？」）
   - **回答**: それに対する答え（例: 「10:00〜19:00です」）
   - **カテゴリ**: 「料金」「営業情報」など、分類のためのラベル（空欄でも可）
   - **料金備考**: 「税込」など補足があれば入力（空欄でも可）
3. 「追加」ボタンをタップすると保存され、一覧画面に戻ります。

【画像: FAQ新規追加フォームの入力画面】

### 1-2. FAQを編集する

1. 管理画面トップのFAQ一覧から、直したいFAQの「編集」ボタンをタップします。
2. 内容を書き換えます。
3. 「保存」ボタンをタップします。

【画像: FAQ編集フォームの画面】

### 1-3. FAQを無効化する（削除ではありません）

1. 一覧の該当FAQにある「無効化」ボタンをタップします。
2. 確認画面で「OK」を選ぶと、一覧から消えます。

> 補足: 無効化してもデータそのものは消えず、bot（自動返信）が参照しなくなるだけです。
> 間違えて無効化した場合は、開発担当者にご連絡ください（現時点では画面から元に戻す機能はありません）。

【画像: 無効化前の確認ダイアログ】

## 2. メニュー・料金の更新

「メニュー・料金を管理する」画面では、施術メニューの名前・料金・施術時間・備考を管理できます。
ここに登録した内容も、bot の自動返信に使われます。

### 2-1. メニューを追加する

1. 管理画面トップの「メニュー・料金を管理する」ボタンをタップします。
2. 「新規追加」ボタンをタップします。
3. 次の4つを入力します。
   - **メニュー名**: 例）カット、パーマ、カラー
   - **料金（円）**: 半角数字のみ（例: 4500）
   - **施術時間（分）**: 半角数字のみ（例: 60）
   - **備考**: 「税込」など（空欄でも可）
4. 「追加」ボタンをタップします。

【画像: メニュー新規追加フォームの画面】

### 2-2. メニューを編集する／無効化する

FAQと同じ操作です。一覧の「編集」または「無効化」ボタンをタップしてください。

【画像: メニュー一覧画面（名前・料金・施術時間が並んだリスト）】

## 3. お知らせを送る（一斉配信）

友だち登録している方全員に、お知らせを一斉に送信できます。
**送信すると取り消せません。** 内容をよく確認してから送信してください。

1. 管理画面トップの「お知らせを送る」ボタンをタップします。
2. テキスト欄にお知らせ内容を入力します（500文字まで、画面下に残り文字数が出ます）。
3. 「送信する」ボタンをタップします。
4. 「友だち全員にお知らせを送信します。よろしいですか？」という確認が出るので、内容に間違いがなければ「OK」を選びます。
5. 送信が完了すると「送信しました」と表示されます。

【画像: お知らせ入力画面（文字数カウント・送信するボタンが見える状態）】
【画像: 送信前の確認ダイアログ】
【画像: 送信完了メッセージ】

> 注意: 誤字脱字がないか、送信前に必ず読み返してください。
> 迷ったら、まず自分（オーナー）のLINEに近い内容を試し送りできる仕組みは現時点ではないため、慎重に確認をお願いします。

## 4. botが変な返答をした時の対処法

AIが自動で答える仕組み上、まれに期待と違う返答をすることがあります。

### まず確認すること

1. 「会話ログを見る」画面を開き、どんな質問にどう答えたかを確認します。
   - **赤色のバッジ**が付いている会話は「未対応（うまく答えられなかった質問）」です。
   - オーナーのLINEにも、未対応の質問があったときに通知が届きます。

【画像: 会話ログ画面（赤バッジの未対応ログとグレーバッジの対応済みログ）】

### 直し方

- **FAQに載っていない内容を聞かれて答えられなかった場合**
  → 「1. FAQの追加」の手順で、その質問と正しい回答をFAQに追加してください。次回から正しく答えられるようになります。

- **メニューや料金について間違った返答をした場合**
  → 「2. メニュー・料金の更新」で、該当メニューの内容が最新か確認・修正してください。

- **FAQを直してもまだおかしい、様子がおかしい場合**
  → 開発担当者に会話ログの内容（いつ・どんな質問・どんな返答だったか）を伝えてご連絡ください。
